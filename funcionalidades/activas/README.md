# Funcionalidades Activas

Esta carpeta contiene las especificaciones de funcionalidades en desarrollo activo.

## Cómo usar

1. Mueve aquí los archivos desde `../pendiente/` cuando inicies el desarrollo
2. Actualiza el archivo marcando las tareas completadas
3. Trabaja en bloques testables (completa backend + frontend + testing juntos)
4. Cuando termines, mueve el archivo a `../completadas/`

## Principios

- **Bloques testables**: Cada tarea debe poder probarse independientemente
- **Progreso incremental**: Marca las tareas a medida que las completes
- **Enfoque**: Trabaja en una funcionalidad a la vez antes de empezar otra
