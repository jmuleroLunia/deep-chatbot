# Funcionalidades Completadas

Esta carpeta contiene las especificaciones de funcionalidades completadas y probadas.

## Cómo usar

1. Mueve aquí los archivos desde `../activas/` cuando completes la funcionalidad
2. Asegúrate de que todas las casillas estén marcadas
3. Añade la fecha de completado al inicio del archivo

## Formato de archivo completado

```markdown
**Completado**: YYYY-MM-DD

# [Nombre de la Funcionalidad]

## Description
...

## Tasks
### Task 1: [Nombre]
- [x] Backend: ...
- [x] Frontend: ...
- [x] Testing: ...

...
```
